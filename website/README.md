# The website for CascadiaJS 2016

[2016.cascadiajs.com](http://2016.cascadiajs.com)

Powered by [surge](http://surge.sh/).

To check out:

```sh
git clone https://github.com/cascadiajs/2016.cascadiajs.com.git
```

To get running:

```sh
npm install
npm start
# Visit http://localhost:9000
```

To deploy:

```sh
npm run deploy
```
To stage:

```sh
npm run stage
```
